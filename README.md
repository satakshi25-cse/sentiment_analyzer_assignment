# Sentiment Analysis of Restaurant Reviews (Majitar)

##  Project Overview
This project analyzes customer reviews of restaurants in Majitar using Machine Learning.

##  Technologies Used
- Python
- Scikit-learn
- NLTK

##  Features
- Sentiment classification (Positive, Negative, Neutral)
- Restaurant-wise sentiment analysis
- Model evaluation (Accuracy, Precision, Recall, F1-score)

##  Project Structure
- sentiment_project.py → Main code
- reviews.csv → Dataset
- screenshots/ → Output images
- Report & PPT included

##  How to Run
pip install pandas nltk scikit-learn  
python sentiment_project.py

##  Output
The model predicts sentiment and provides overall restaurant rating.

##  Future Scope
- Real-time Google review integration
- Deep learning models (BERT)
